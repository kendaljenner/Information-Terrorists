## Archiving Instructions

### Method 1: Wayback Machine (web.archive.org)

1. Visit https://web.archive.org/
2. Enter URL in "Save Page Now" box
3. Click "Save Page"
4. Wait for capture to complete
5. Record resulting archive URL in this document

### Method 2: Archive.today

1. Visit https://archive.today/
2. Enter URL
3. Submit for archiving
4. Record resulting URL

### Method 3: Direct GitHub Cloning (for repositories)

```bash
git clone https://github.com/kjedrdev/Data_Poisoning.git
git clone https://github.com/kjedrdev/NWO-KI_Elite.git
```

### Method 4: wget Mirror (for complete site backup)

```bash
wget --mirror --convert-links --adjust-extension --page-requisites --no-parent [URL]
```
