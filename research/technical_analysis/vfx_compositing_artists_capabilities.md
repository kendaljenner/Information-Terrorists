## 2. VFX Compositing: Technical Overview

### What is VFX Compositing?

**Definition:**
- Combining visual elements from separate sources into single images
- Creating seamless integration of real and computer-generated footage
- **Pixel-perfect results**
- Industry standard for film/TV

**Core Techniques:**

#### 2.1 Node-Based Compositing (Nuke/NukeX)

**Industry Standard:**
- **Nuke** - Foundry (primary industry tool)
- **NukeX** - Extended version
- **Node-based workflow:** Visual programming for VFX

**Capabilities:**
- Multi-channel workflows (RGBA + data passes)
- 3D compositing integration
- Deep compositing
- Stereoscopic workflows
- **Resolution independent**

**Professional Mastery Required:**
- Node graph optimization
- Multi-pass rendering workflows
- Color space management (ACES, etc.)
- Pipeline integration (Python scripting)

#### 2.2 Tracking and Match-Moving

**Techniques:**
- **2D Tracking:** Feature tracking, planar tracking
- **3D Camera Tracking:** Match-moving, camera solve
- **Object Tracking:** Tracking moving objects
- **Planar Tracking:** Mocha, After Effects, Nuke

**Group Application:**
- Match deepfake elements to real footage
- Maintain perspective consistency
- **Seamless integration**

#### 2.3 Rotoscoping

**Definition:**
- Frame-by-frame isolation of elements
- Manual or assisted masking
- Hair/fur detail work
- **Labor-intensive, skill-intensive**

**Professional Level:**
- Clean edges
- Motion blur handling
- Detail preservation
- **Forensic-level precision**

#### 2.4 Color Science and Grading

**Technical Requirements:**
- Color space conversion (sRGB, Rec.709, DCI-P3, ACES)
- LUT creation and application
- Shot-to-shot matching
- HDR workflows

**Group Application:**
- Match deepfake lighting to real footage
- Consistent color across fabricated content
- **Indistinguishable from real**

#### 2.5 Keying and Extraction

**Techniques:**
- Green/blue screen keying
- Luma keying
- Difference keying
- **Clean subject isolation**

**Professional Mastery:**
- Edge detail preservation
- Spill suppression
- Transparency handling
- **No artifacts**
